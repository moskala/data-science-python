def plot_2D(X, k, labels):
    """
    Funkcja przyjmuje dwuwymiarow macierz X o wymiarach (n,2) i wektor etykiet, 
    gdzie labels[i] to etykieta X[i] punktu.
    Funkcja rysuje wykres 2D punktw z X z podziaem na kolory wedug odpowiadajcych im etykiet.
    Funckja jest uywana jedynie w tym skrypcie, wic zakadana jest poprawno danych wejciowych.
    """
    
    colors = cycle('bgrcmyk')
    for k, col in zip(range(k), colors):
        class_members = labels == k
        plt.scatter(X[class_members, 0], X[class_members, 1], c=col)

    plt.title('Analiza skupie')
    plt.show()
    
def plot_3D(X, k, labels):
    """
    Funkcja przyjmuje dwuwymiarow macierz X o wymiarach (n,3) i wektor etykiet, 
    gdzie labels[i] to etykieta X[i] punktu.
    Funkcja rysuje wykres 3D punktw z X z podziaem na kolory wedug odpowiadajcych im etykiet.
    Funkcja jest uywana jedynie w tym skrypcie, wic zakadana jest poprawno danych wejciowych.
    """
    
    colors = cycle('bgrcmyk')
    
    fig = plt.figure()
    ax = fig.add_subplot(111, projection='3d')
        
    for k, col in zip(range(k), colors):
        class_members = labels == k      
        ax.scatter(xs=X[class_members, 0], ys=X[class_members, 1], zs=X[class_members, 2], c=col)

    ax.set_xlabel('X Label')
    ax.set_ylabel('Y Label')
    ax.set_zlabel('Z Label')


    plt.title('Analiza skupie')
    plt.show()

def show_input(X, title):
    """
    Funkcja przedstawia punkty macierzy X na wykresie 2 lub 3 wymiarowym dodajc tytu title.
    Funckja jest uywana jedynie w tym skrypcie, wic zakadana jest poprawno danych wejciowych.
    """
    
    if X.shape[1] == 2:
        plt.scatter(X[:, 0], X[:, 1])
        plt.title(title)
        plt.show()
    elif X.shape[1] == 3:
        fig = plt.figure()
        ax = fig.add_subplot(111, projection='3d')   
        
        ax.scatter(xs=X[:, 0], ys=X[:, 1], zs=X[:, 2])

        ax.set_xlabel('X Label')
        ax.set_ylabel('Y Label')
        ax.set_zlabel('Z Label')

        plt.title(title)
        plt.show()

Testy poprawności zaimplementowanych metod¶

Do przeprowadzenia testw poprawnoci wykorzystano funkcj adjusted_rand_score z pakietu sklearn.metrics.cluster.

In [7]:
from sklearn.metrics.cluster import adjusted_rand_score

Test 1¶

Za pomoc wielowymiarowego rozkadu normalnego tworzone s dwa skupiska punktw w przestrzeni R2.

In [8]:
pts = 50
a = np.random.multivariate_normal([0, 0], [[10, 1], [1, 10]], size=pts)
b = np.random.multivariate_normal([20, 10], [[10, 2], [2, 10]], size=pts)
X = np.vstack((a, b))

Rozmieszczenie punktw ze zbioru X prezentuje poniszy wykres:

In [9]:
show_input(X,'Zbir danych dwuwymiarowych')

Analizujc powyszy wykres, moemy przypisa dwie etykiety do tego zbioru danych:

* skupisko o wartoci redniej [0,0]
* skupisko o wartoci redniej [20,10]

Dla kadego punktu przypisujemy odpowiadajc mu referencyjn etykiet, biorc pod uwag z ktrego rozkadu zosta otrzymany:

In [10]:
label_a = np.array([0] * pts)
label_b = np.array([1] * pts)
labels_true = np.hstack((label_a, label_b))

Na tak przygotowanych danych wykonujemy zaimplementowany algorytm spektralny:

In [11]:
k = 2
labels_pred = spectral_clustering(X, k, 5)
plot_2D(X, k, labels_pred)

Z powyszego wykresu, moemy wnioskowa, e obserwacje zostay poprawnie przydzielone do etykiet. Jednak do oceny jakoci podziau moemy posuy si skorygowanym indeksem Randa z biblioteki sklearn.metrics:

In [12]:
adjusted_rand_score(labels_true, labels_pred)
Out[12]:
1.0

Otrzymany wynik 1.0 oznacza, e wszystkie wyznaczone etykiety zgadzaj si z referencyjnymi, zatem algorytm analizy skupie jest dobry.

Podsumowanie¶

Testy zostay przeprowadzone na zbiorach danych w przestrzeniach R2, R3, R4.

Dla kadego testu, otrzymane wartoci miar podobiestwa k-podziaw (referencyjnego oraz wygenerowanego) byy bliskie 1.0.

Zatem ocena jakoci podziau wykazaa, e zaimplementowany algorytm spektralny jest dobry, to znaczy generuje podziau podobne do referencyjnych etykiet.
